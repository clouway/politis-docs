# Мобилно приложение — преглед

Politis Mobile е Android приложението, което търговският представител използва на терен. Поддържа pre-sales (заявка → доставка) и van-sales (продажба на място), работи с баркод скенер, GPS координати на клиентските обекти и има пълна офлайн поддръжка.
